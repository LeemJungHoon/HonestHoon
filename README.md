# HonestHoon

HonestHoon 홈페이지 제작

Feat : 새로운 기능 추가(온전한 기능으로 동작할 경우에만. Feat이후의 코드 손질은 fix나 style, 혹은 refactor 방향으로 가야함)
Fix : 버그 수정
Docs : 문서 수정(ReadME 파일과 같이 문서 수정인 경우)
Style : 코드 포맷 변경 (동작에 영향 없음. CSS수정인 경우)
Refactor : 코드 리팩토링 (겉으로 드러나는(사용자가 봤을 때) 변화는 없는 상태)
Test : 테스트 추가 또는 수정
Chore : 작업 진행중. Docs같이 문서 수정은 아니고 코드를 만졌으나 기능이 아직 완전히 추가된 Feat 상태는 아니고
CSS수정인 Style수정도 아니면서 버그 수정인 Fix도 아닌 경우
